5. Recommendations

  1. Network Segmentation: Enforce Purdue model zone boundaries with firewalls and unidirectional gateways.
  2. Access Control: Implement RBAC for all ICS protocol access. Disable default credentials.
  3. Monitoring: Deploy continuous passive monitoring with ICS-aware IDS rules.
  4. Patch Management: Address critical CVEs on PLCs, RTUs, and HMIs per vendor advisories.
  5. Authentication: Enable authentication on Modbus, S7comm, and EtherNet/IP where supported.
